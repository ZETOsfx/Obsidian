# Теория сравнений 

Целые числа A и B сравнимы по модулю M, если имеют одинаковые остатки от деления на M. Запись: 
`a = b mod(m); m > 0; m - цел.`
`~ a - b = mt`

Свойства сравнений:

1.
1) a = c (mod m)
2) b = c (mod m)
=> a = b (mod m)

2.
1) ai = bi (mod m)
=> ∑ai = ∑bi (mod m)

2a.
1) a = b (mod m)
=> ak = bk (mod m)

3.
1) ai = bi (mod m)
=> Пai = Пbi (mod m)

3a.
1) ai = bi (mod m)
=> a^k = b^k (mod m)

4.
1) a = b (mod m)
=> ak = bk (mod km)
